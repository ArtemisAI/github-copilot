# VS Code Remote Development (microsoft/vscode-remote-release)

## Overview

Visual Studio Code Remote Development: Open any folder in WSL, in a Docker container, or on a remote machine using SSH and take advantage of VS Code's full feature set.

This repository provides feedback on the Visual Studio Remote Development extension pack and its related extensions.

## Extensions Included

The extension pack includes:

- **Remote - SSH**: Editing Configuration Files, Remote - SSH: Editing Configuration Files
- **Dev Containers**: Developing inside a Container
- **WSL**: Developing in WSL
- **Remote - Tunnels**: Access to Visual Studio Code Server

## Releases

VS Code Remote Development extension "stable" releases are tied directly to VS Code releases. Release highlights can be found in VS Code release notes which will include a link to detailed extension release notes.

The extensions are developed using the same development process and schedule as VS Code itself. You can see what is planned for the current development iteration in a pinned planning issue in this repository.
